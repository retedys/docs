## What are boards?

Boards are branded collections of policies (e.g. "Onboarding Essentials", "Security & Privacy", "HR Basics"). They control which employees see which policies and how they're presented.

Board status: **Draft** → **Published** → **Archived**

## Create a board

1. **Boards** → **New Board**
2. Add title and description
3. Branding (optional): logo, colours, cover image
4. Add policies via drag-and-drop
5. Assign audience (groups, users)
6. Publish when ready

## Board settings

| Setting | Purpose |
|---------|---------|
| Title, description | Identity |
| Logo, colours, background | Branding for employee view |
| Policy list | Order policies via drag-and-drop |
| Audience | Groups and object-level shares |

## Object-level sharing

Share a board with specific users at different levels: **owner**, **admin**, **editor**, **assignee**, **viewer**.
